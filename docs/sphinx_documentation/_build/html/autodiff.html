get_derivative(point: Union[int, float, list, numpy.ndarray], seed_vector=None, mode='forward')[source]¶

calculate the directional derivative given point and the seed vector

Parameters

  • point (int, float, list, or numpy ndarray) – a single or a sequence of numbers defining the point for the functions to evaluate at

  • seed_vector (int, float, list, or numpy ndarray) – a single or a sequence of numbers defining the seed of direction

  • mode ({"forward", "f", "reverse", "r"}) – option to perform automatic differentiation using forward or reverse mode; default is “forward”

Returns

the directional derivative based on the seed

Return type

int, float, or numpy ndarray

Raises

  • TypeError – If point or seed_vector is not an int, float, list, or numpy ndarray or has incorrect dimension

  • ValueError – If mode is not one of the accepted strings
